🌸 CYF Step 6 Project: The Future of Women in Coding
Project Overview

This project was created as part of the CodeYourFuture (CYF) Step 6 assignment. It is a simple, responsive website built using HTML and CSS. The purpose of the project is to demonstrate basic front-end development skills while sharing an important message about women in technology.

The website focuses on the theme “The Future of Women in Coding”, highlighting opportunity, inclusion, and the growing role of women in the tech industry.

📖 Background

The future of women in coding is full of opportunity and growth. Technology is becoming one of the fastest-growing fields in the world, and women are playing an important role in shaping it.

Women belong in technology because they bring different perspectives, creativity, and problem-solving skills that help build better and more inclusive systems. Coding is not only about writing software, but also about creating solutions that can improve lives, businesses, and communities.

In the future, more women will work in areas such as artificial intelligence, web development, cybersecurity, and digital entrepreneurship. As technology continues to evolve, it is important that women are equally represented in shaping it.

I am learning coding because I want to be part of this future and use technology to create opportunities and make a positive impact.
